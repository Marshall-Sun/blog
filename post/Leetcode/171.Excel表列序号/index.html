Leetcode/171.Excel表列序号
发表于 2020-09-19
字数总计:193|阅读时长: 1 分钟

问题描述

给定一个Excel表格中的列名称，返回其相应的列序号。

例如，

A -> 1
B -> 2
C -> 3
...
Z -> 26
AA -> 27
AB -> 28 
...

示例 1:

输入: "A"
输出: 1

示例 2:

输入: "AB"
输出: 28

示例 3:

输入: "ZY"
输出: 701
展开

代码

解法

转换成列表，遍历相加即可
