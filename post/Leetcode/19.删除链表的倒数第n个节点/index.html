Leetcode/19.删除链表的倒数第n个节点
发表于 2020-08-31| 更新于 2020-10-14
字数总计:252|阅读时长: 1 分钟

问题描述

给定一个链表，删除链表的倒数第 n 个节点，并且返回链表的头结点。

示例：

给定一个链表: 1->2->3->4->5, 和 n = 2.

当删除了倒数第二个节点后，链表变为 1->2->3->5.
说明：

给定的 n 保证是有效的。

进阶：

你能尝试使用一趟扫描实现吗？

展开

代码

解法

  1. 在while循环中，遍历链表将fast指向尾部
  2. 利用链表长度 - n = 目标节点编号将第二个指针slow指向目标节点的父节点
  3. 判断特殊情况，是否删除的是首尾节点
