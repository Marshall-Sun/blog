字数总计:6.6k|阅读时长: 24 分钟

实验目的

在基础实验中，能熟练运用Python语言的基本语法、控制语句、函数、面向对象、GUI等开发一个应用类游戏。为后续Python学习打下扎实基础。

在数据分析实验中，能熟练运用Python列表、字典、集合、数组等数据结构以及索引和切片等查询操作解决实际问题，熟练掌握Python读写数据文件的方法，熟练使用Matplotlib数据可视化工具，熟练使用pandas数据分析库函数，基本掌握使用scikit-learn库多种机器学习算法进行数据分析的过程，了解Python程序的调试方法，运用Python编写程序解决实际应用问题。

实验内容

1. 基础实验

设计一个带有图形用户界面的人机对战井字棋游戏。

游戏在九宫方格内进行，如果一方抢先于某方向（横、竖、斜）连成3子，则获取胜利。游戏界面，首先询问哪个玩家先走，然后根据玩家落子位置，显示棋盘状态。最终根据游戏规则，评判哪个玩家获胜。

2. 数据分析实验

  1. 分析1996—2015年人口数据特征间的关系以及各个特征的分布与分散状况
  2. 读取并查看P2P网络贷款数据主表的基本信息，使用分组聚合方法进一步分析用户信息更新表和登录信息表
  3. 插补用户用电量数据缺失值，合并线损、用电量趋势与线路告警数据，标准化建模专家样本数据
  4. 使用sklearn处理wine和wine_quality数据集，构建基于wine数据集的K-Means聚类模型、SVM分类模型、回归模型

实验环境

  • Python 3.7.4 64-bit
  • Anaconda 2019.10
  • Visual Studio Code 1.40.2

实验过程与分析

基础实验

1. 问题描述及分析

本实验要求设计一个带有图形用户界面的人机对战井字棋游戏。故采用tkinter来创建GUI程序。在游戏界面中，首先通过tkinter的messagebox弹窗询问哪个玩家先走，然后根据玩家落子位置，显示棋盘状态。最终根据游戏规则，再次调用messagebox显示游戏结果。

2. 关键算法设计

创建了一个列表buttons[]来初始化存储井字棋中所有的9个按键，并将其绑定到界面上。同时当用户点按按钮时，调用btnClick()函数来检测此按钮是否可点击，若不可点击则弹窗警告；若可点击则调用checkForWin()函数检测当前情况是否结束游戏，若游戏已结束则弹窗显示游戏结果；若游戏继续，则调用computer_move()函数来计算电脑的下一步棋，并将选择权交回给用户，周而复始。

数据分析实验

1. 分析数据特征间的关系以及各个特征的分布与分散状况

（1）问题理解

本实验要求分析1996至2015年人口数据特征间的关系以及各个特征的分布与分散状况。故采用numpy来读取人口数据，同时使用matplotlib.pyplot模块来进行图像的绘制。通过对数据内容的分析，绘画出散点图、折线图、柱状图、饼图、箱线图，进而可较为直观地了解1996至2015年人口数据的大致情况。

（2）实验过程

通过numpy的load()方法来实现实验数据的读取，并使用pyplot中提供的对应绘图函数构建五个模型，分别为1996至2015年人口散点与折线图、1996至2015年男女人口与城乡人口对比柱状图、1997至2015年全国人口变化折线图、1996与2015年男女人口与城乡人口对比饼图与1996至2015年人口数据箱线图。经过多次分析测试，各图都达到了较好的展示效果，易于从图中获得相关信息，达到了构建模型的目的。

（3）实验结果分析

  1. 1996至2015年人口散点与折线图如下，根据此图，可以大致地了解年末总人口、男性人口、女性人口、城镇人口与乡村人口的历年分布情况。而通过对这张图的绘制，初步掌握了使用pyplot来绘制图片的方法，同时也根据实际需要将横纵轴信息优化地更为美观。

  2. 1996至2015年男女人口与城乡人口对比柱状图如下，根据此图，可直观地了解男女人口与城乡人口的历年变化情况：从1996年一直到2015年，男性人口数一直多于女性人口数，且逐渐保持一个相对稳定的差值；而城镇人口与乡村人口的情况则恰恰相反，城镇人口在逐年增加，乡村人口在逐年减少，在2011年城镇人口完成了对乡村人口的数量反超。

  1. 1997至2015年全国人口变化折线图如下，通过此图，可直观地了解全国历年的年末总人口变化详细情况。可以看出，在2009年之前全国年末总人口在逐年上升的同时，人口增长率却在逐年下降，可以容易得出这是计划生育政策的影响。在2009年之后人口增长率基本保持不变，但由于我国人口基数大，全国年末总人口仍保持上升趋势。

  2. 1996与2015年男女人口与城乡人口对比饼图如下，通过此图，可以大致地了解男女人口以及城乡人口从1996年开始直至2015年结束的这一段时间中各部分占总体比例的变化情况。由图可知，从1996年至2015年，男性人口与女性人口的比例几乎无较大变化，从微观上看，男性比例稍微多于女性比例，但在宏观上看，男性比例于女性比例大致相等。而城乡人口比例在同一段时期则变化较大，城镇人口从1996年占比不到三分之一逐渐增多，至2015年占比几乎为总人口的三分之二，于此相伴的是乡村人口占比的减少，这一现象直接反映出我国城市化现象明显。

  3. 1996至2015年人口数据箱线图如下，通过此图，可以大致地了解年末总人口、男性人口、女性人口、城镇人口与乡村人口的数据分散情况。因为年末总人口是各类型人口数量之和，其箱线图相对来说位于高处。对于男性人口与女性人口，可以看出两者的变化都不大。而城镇人口与乡村人口的历年变化程度是最大的。

（3）实验结果分析

在实验过程中得到了非常多的有效信息。首先，在查看维度、大小、占用内存信息的过程中，发现登录信息表、用户信息更新表与主表的所含数据都极大，登录信息表与用户信息更新表都包含着十万级别的用户数据，但其数据内容较少，每一行数据仅包含四五项内容；但是主表的情况则有所区别，其仅含三万行用户数据，相对较少，然而每一行数据包含了上百条内容，这对接下来的数据处理工作提出了挑战。

在数据处理过程中，首先清除无效数据，即通过dropNullStd()将值相同或全为空的列剔除，然后使用describe方法进行描述性统计，这样对于表中的每一列数据都可以看出其平均数、总数、中位数、总数与上下四分位点。然后通过转换并提取用户信息更新表和登录信息表的时间字符串，计算用户信息更新表和登录信息表中两时间的差，可以得出各用户登录至更新信息的延迟时间。最后通过分组可以了解到不同用户的最早和最晚更新及登录时间以及登录次数。

（3）实验结果分析

本实验完成了处理数据这一任务，得到了一定量的信息。首先，在缺失值插补的过程中，了解到三个用户用电量的基本情况，并成功使用SciPy库中interpolate模块中的lagrange()方法对数据进行拉格朗日插值，补全了缺失的数据。其次，在内连接数据方面，采用pandas的merge()方法将ele_loss.csv和alarm.csv表分别按ID和date两个键值作为主键进行内连接，获得了数据链接后的DataFrame对象。最后通过自定义标准差标准化函数StandardScaler()，分别对3列数据进行标准化，查看到标准化后的数据，极大地方便了后续数据建模操作。

4. 数据分析

（1）问题理解

本实验要求使用sklearn处理wine和wine_quality
数据集，构建基于wine数据集的K-Means聚类模型、SVM分类模型、回归模型。根据实验具体要求，需充分学习与熟练掌握sklearn的用法，同时借助sklearn的各项功能对实验所提供的数据集进行划分、降维、建模与评价。

（3）实验结果分析

在本实验中，首先从对应csv文件读取wine数据集和wine_quality数据集并对其拆分、降维，掌握了初步处理这些数据的方法，为接下来的工作做好了准备。其次，先算出区间由2至10的聚类模型的FMI指数和Calinski-Harabasz指数，加以比较后发现聚类数目为2或3时其指数较大，这代表着类自身更加紧密，类与类之间越分散，即更优的聚类结果。 因此进一步绘制出聚类数目为2的K-Means模型以及聚类数目为3的K-Means模型散点图，由两个散点图看出聚类数目为3时散点更加分散，即聚类结果最好。

接下来，通过调用实验开始时划分的训练集和测试集，调用sklearn中的MinMaxScaler()和transform()函数进行离差标准化，得到标准化后的数据集wine_trainScaler和wine_testScaler。这样，就可以构建SVM模型，并得到预测测试集结果与分类报告，由报告可以直观地得到各值的精确度，召回率，F1值以及出现次数。我们也可以通过构建好的SVM模型来绘制接受者操作特征曲线(ROC)，因为ROC曲线越靠近左上角，诊断价值越大；ROC曲线下面积越大，应用价值越大，所以可认为此SVM模型分为2类和3类的价值都较大。

最后，采用之前处理好的wine_quality数据集构建线性回归模型、梯度提升回归模型，结合真实评分和预测评分，计算均方误差、中值绝对误差、可解释方差值。由这十个方差值可以看出，wine_quality数据集的线性回归模型和梯度提升回归模型总体性能较为接近，在不同方差值上各有千秋，因此根据两个模型分别绘制折线图。从两幅折线图中可看出，梯度提升回归模型与线性回归模型的预测效果相差无几，因此认为这两个模型的预测效果相似，需结合特定需求来进行模型的选用。

实验总结

本系列实验包括一个基础实验以及四个数据分析实验。在基础实验中使用了tkinter来绘制GUI界面，完成了一个井字棋程序。我认为其主要的不足在于界面不够现代，不能吸引用户持续游玩，而导致如此的原因一是tkinter的各种控件不够现代，二是实验时间、实验精力不足使得引用自定义美化资源过少，希望在今后的实验中能更进一步。

而在数据分析实验一中，使用了pyplot来分析1996—2015年人口数据特征间的关系以及各个特征的分布与分散状况。我认为其主要的不足是我对pyplot的主要功能掌握地不够熟练，导致绘图过程中走了很多弯路，这启示着要善于学习与搜索资源，这样在学习新知识时才能如鱼得水。在数据分析实验二中通过对pandas的使用，成功读取并查看P2P网络贷款数据主表的基本信息，使用分组聚合方法进一步分析用户信息更新表和登录信息表。我认为此实验不足之处主要为pandas的各种功能较为复杂，其中DataFrame的功能又与python自带的列表功能有较大出入，我在处理DataFrame中各种数据时效率过低，这也应归咎于我自身的知识储备不够完善，所幸最后通过搜索引擎解决了一系列的问题。

在数据分析实验三中，使用了SciPy库中interpolate模块中的lagrange()方法插补用户用电量数据缺失值，并利用pandas合并线损、用电量趋势与线路告警数据，标准化建模专家样本数据。在数据分析实验四中则采用sklearn处理wine和wine_quality数据集，构建基于wine数据集的K-Means聚类模型、SVM分类模型、回归模型并通过绘制相关图表来直观地展示模型拟合情况，得出结论。我认为这两个实验在实验设计上都非常详细，在实验要求上贴合所学知识，在实验目的上要求明确。我在完成实验的过程中虽然遇到了一些挫折，但最终将其逐一解决，因此我认为这两个实验无不足之处。
