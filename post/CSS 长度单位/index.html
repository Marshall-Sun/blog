CSS 长度单位
发表于 2020-11-06| 更新于 2020-11-15|前端知识CSS
字数总计:288|阅读时长: 1 分钟

长度单位有两种：绝对单位和相对单位。

绝对单位

绝对长度单位是固定的值，不建议在已知变化的屏幕尺寸上使用绝对长度单位。

绝对单位 描述
cm centimeters 厘米
mm millimeters 毫米
in inches (1 in = 2.54 cm) 英尺
pc picas (1 pc = 1/6 in) 派卡，印刷行业使用
pt points (1 pt = 1/72 in) 逻辑像素
px pixels (1 px = 1/96 in) 物理像素

注意：物理像素（px）在不同的设备有不同表现。对于低分辨率屏幕，1px 表示一个逻辑像素，对于高分辨率屏幕，1px 表示多个逻辑像素。

相对单位 描述
em 相对于当前元素字体“M”宽度
ex 相对于当前元素字体“M”高度，很少用到
ch 相对于当前元素字体“0”宽度，很少用到
rem 相对于根元素字体“M”宽度
vw 视口宽度 window.innerWidth 的百分比
vh 视口高度 window.innerHeight 的百分比
vmin 取当前 vw 和 vh 中最小值
vmax 取当前 vw 和 vh 中最大值
% 相对于父元素对应长度的百分比
