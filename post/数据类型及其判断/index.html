数据类型及其判断
发表于 2020-11-14| 更新于 2020-11-16|前端知识JavaScript
字数总计:934|阅读时长: 3 分钟

数据类型种类

八种，String, Number, Boolean, Object, Undefined, Null, Symbol, BigInt。

Symbol 代表创建后独一无二且不可变的数据类型，它主要是为了解决可能的全局变量冲突。

BigInt 是一种数字类型数据，它可以表示任意精度的整数，故可以安全地存储和操作超出 Number 范围 [-(2^53 - 1), 2^53 - 1] 的整数。

Undefined 与 Null 的具体区别可以参考 关于 Undefined 与 Null。

数据类型存储

在数据结构中，栈中数据的存取方式为先进后出。而堆是一个优先队列，按优先级来进行排序。完全二叉树是堆的一种实现方式。

在操作系统中，内存被分为栈区内存和堆区内存。

  • 栈区内存由编译器自动分配释放，存放函数的参数值，局部变量的值等。其操作方式类似于数据结构中的栈。

  • 堆区内存一般由程序员分配释放，若程序员不释放，程序结束时可能由垃圾回收机制回收。

栈储存着原始数据类型，共七种：String, Number, Boolean, Undefined, Null, Symbol, BigInt。存储占据空间小、大小固定，属于被频繁使用数据，所以放入栈中存储。

堆储存着引用数据类型，共一种：Object。存储占据空间大、大小不固定。如果存储在栈中，将会影响程序运行的性能；引用数据类型在栈中存储了指针，该指针指向堆中该实体的起始地址。当解释器寻找引用值时，会首先检索其在栈中的地址，取得地址后从堆中获得实体。

判断

typeof

typeof 是 JavaScript 的一元运算符，它返回一个字符串，表示未经计算的操作数的类型。

被判断类型 结果
Undefined "undefined"
Null "object"
Boolean "boolean"
Number "number"
BigInt "bigint"
String "string"
Symbol "symbol"
Function 对象 "function"
浏览器提供的对象 取决于具体实现
其他任何对象 "object"
