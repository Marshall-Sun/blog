定义

Ecma 国际（Ecma International）是一家制定并维护计算机各种标准的国际组织。前名为欧洲计算机制造商协会（European Computer Manufacturers Association，ECMA）。

TC39 是 Ecma 国际的一个部门，专门负责制定 ECMA 第 262 号标准，标准代号为 ECMA-262，这个标准就是 ECMAScript。

JavaScript 则是一种通用脚本编程语言，它遵循了 ECMAScript 标准。换句话说，JavaScript 是 ECMAScript 的一种方言。

剩余参数

剩余参数（Rest parameters）语法允许将一个不定数量的参数表示为一个数组。

如果函数的最后一个命名参数以 ... 为前缀，则它将成为一个由剩余参数组成的真数组。

和 arguments 对象之间的区别：

  • 剩余参数只包含那些没有对应形参的实参，而 arguments 对象包含了传给函数的所有实参。
  • arguments 对象不是一个真正的数组，而剩余参数是真正的 Array 实例，可以直接使用所有的数组方法。
  • arguments 对象还有一些附加的属性。

let 与 const

ES2015（ES6）之前，声明变量的方法只有 var。let 与 const 的加入引入了块级作用域的概念，见 作用域与闭包。

它们的区别：

  • let 和 const 声明形成块作用域。
  • 同一作用域下 let 和 const 不能声明同名变量，而 var 可以。
  • var 声明的变量会挂载在 window 上，而 let 和 const 声明的变量不会。
  • var 声明变量存在变量提升，let 和 const 不存在变量提升，如果在声明之前使用这些变量就会报错，这就是暂时性死区（temporal dead zone）。
  • const 一旦声明必须赋值，不能使用 null 占位。原始数据类型声明后不能再修改，引用数据类型可以修改其属性。

ES2017（ES8）新特性

async 函数

见 异步编程。

Object.values()

返回一个数组，包括给定对象的所有可枚举属性值。

注意：Object.keys() 返回的是给定对象的所有属性键名，注意区分。

Object.entries()

返回一个数组，包括给定对象的所有可枚举属性键值对数组。

Object.getOwnPropertyDescriptors()

用来获取一个对象的所有自身属性的描述符。如果没有任何自身属性，则返回空对象。
